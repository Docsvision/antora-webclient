==== Скачать дополнительные файлы

Когда свойство активно, скачивание дополнительных файлов разрешено.

==== Скачать основные файлы

Когда свойство активно, скачивание основных файлов разрешено.

==== Скачать только файлы с подписью

Когда свойство активно, скачиваются только файлы с подписью. В зависимости от выполненных для ЭУ настроек свойств будут скачаны основные и/или дополнительные.

include::partial$properties.adoc[tags=stdCss]

==== Формат подписи

Когда свойство `*Скачать только файлы с подписью*` активно, подпись будет скачиваться в выбранном из списка формате: *_.p7s_*, *_.sgn_*, *_.sig_*, *_.sign_*.
